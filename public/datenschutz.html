FNG Status Plugin

Datenschutzerklärung

Die Anwendung "FNG Status Plugin" erhebt keine Nutzerdaten, setzt keine Cookies und schreibt nicht in den Session Storage. Sie dieht lediglich der Darstellung von Server Status Meldungen der FearNixx UG. Die dafür benötigten Daten werden von der Seite https://status.fearnixx.de/ via REST API calls abgefragt.
